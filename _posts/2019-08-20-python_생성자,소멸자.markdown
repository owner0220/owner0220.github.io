# Python

## 생성자, 소멸자

### **생성자 메소드**  **\_\_init\_\_** 

- 객체의 초기화
- 인스턴스 객체가 생성 될 때 가장 먼저 호출 된다.



### **소멸자 메소드** **\_\_del\_\_**

- 객체의 소멸
- 인스턴스 객체의 레퍼런스 카운트가 0이 될때 자동으로 호출 된다.

※ 인스턴스 사용 후 참조카운트가 0이 되면 스스로 소멸한다.
