# Python

## 리스트

### 리스트란?

- 리스트란 값의 나열 '[' 과 ']' 로 감싼다.

- 순서가 있고, **여러 종류의 값을 담을 수 있다.**

  ※ 하나의 리스트 안에 문자, 숫자 둘다 담을 수 있다!!!
  - append()
  - insert()
  - index()
  - count()
  - pop()
  - remove()
  - sort()
  - reverse()



### 선언법

``` python
a = [1,2,3]
a = ['a',1,'b']
```



**※ list(), set(), tuple() 을 이용해 언제든지 서로 변환 될 수 있다!**
