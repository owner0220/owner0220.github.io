# Python

## 파이썬 모듈

### 모듈

- 여러 코드를 묶어 다른 곳에서 재사용 할 수 있는 코드의 모음이다.



### 사용법

```python
#os, math, bs4 모듈을 포함 시킬때 내용
import os
import math
import bs4

#copy 모듈 안에 copy 메서드를 포함
from copy import copy

#As 참조를 다른이름으로 설정 할 때 사용한다.
#os라는 이름대신 window를 쓰겠어~
import os as window
```
