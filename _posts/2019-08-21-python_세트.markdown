# Python

## 세트

### 세트란?

- 수학의 집합과 유사한 형태로 '{' 과 '}'로 감싼다.

  - **& , |** 사용 가능하다.

- 리스트처럼 값의 모임이지만 **순서가 없고 중복을 허용하지 않는다.**

  ※ 이런 특성으로 중복을 제외하는 필터기능으로 많이 사용한다.

  ```python
  c="hello"
  d=set(c)
  
  >>> d
  {'e','h','l','o'}
  ```



### **선언법**

```python
a = {'a','b','c'}
```

**※ list(), set(), tuple() 을 이용해 언제든지 서로 변환 될 수 있다!**
